# Data Analytics Foundations

## Module Overview
This repository demonstrates how data analytics can be taught with
a strong emphasis on interpretation, insight generation, and communication rather
than tools alone.

---

## Intended Learning Outcomes
By the end of this module, students will be able to:
1. Understand different types of data and data quality issues
2. Perform basic data cleaning and preparation
3. Conduct exploratory data analysis (EDA)
4. Identify patterns, trends, and anomalies
5. Communicate insights using visualisations and narratives

---

## Target Audience
- Diploma/Degree level students, Interns, Fresh Data Scientists
- Minimal programming background assumed

---

## Teaching Approach
- Hands-on, example-driven learning
- Realistic datasets
- Focus on *why* before *how*
- Emphasis on decision-making and interpretation

---

## Suggested Weekly Structure
- **Week 1:** Understanding data & problem framing  
- **Week 2:** Data quality, cleaning, and preparation  
- **Week 3:** Exploratory data analysis  
- **Week 4:** Data visualisation principles  
- **Week 5:** Data storytelling and insight communication  

---

## Assessment Ideas
- Individual EDA assignment
- Group presentation on data-driven insights
- Short reflection on assumptions and limitations

---

## Key Teaching Notes
- Encourage students to question results
- Highlight ethical use of data
- Reinforce clarity over complexity
